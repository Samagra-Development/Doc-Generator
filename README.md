## About :open_book:

An easy tool for generating single and bulk documents in several supported formats from templates.

## Dev Setup
```bash

# start all microservices and application(watch mode)
$ bash run.sh start

# stop all microservices
$ bash run.sh stop

# help
$ bash run.sh --help
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
